# **Crafting Skill Editor**

### Simple in-game editor for crafting skill levels (7 Days to Die 2.5+)  
**Toggle:** F8

---

## **Description**

Crafting Skill Editor adds a lightweight in-game interface that allows you to view and modify crafting-related skill levels instantly.  
Designed for testing, sandbox play, and mod development.

---

## **Features**

- Edit all crafting skills in-game  
- Search and filter skills  
- Set exact values or clamp to maximum  
- Apply changes individually or in bulk  
- Reset skills to original values  
- Clean, compact UI  
- Client-side only  

---

## **Controls**

- Toggle UI with a configurable hotkey (default: **F8**)

**Keybind file:**  
`Config/KeyBinding.xml`

---

## **Compatibility**

- 7 Days to Die 2.5+  
- Compatible with vanilla crafting skills  
- Compatible with other magazine-based mods that follow vanilla progression logic  
- Does not modify XML or save files  
- Minimal performance impact
